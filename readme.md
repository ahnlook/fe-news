# 📰 뉴스스탠드

해당 프로젝트를 통해

- 전체 언론사의 뉴스를 볼 수 있다.
- 언론사별 뉴스를 모아볼 수 있다.
- 언론사를 구독하거나 취소할 수 있다.
- 구독한 언론사 또한 따로 모아서 볼 수 있다.
- 뉴스는 리스트 모드와 그리드 모드로 확인할 수 있다.
